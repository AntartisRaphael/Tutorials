## Supplementary Material

This repository provides supplementary material accompanying the study presented in the poster.

### Contents

- **Experimental Results**
  - Performance metrics for each model and task  
  - Results reported across multiple random seeds  

- **Evaluation Metrics**
  - Overall AUC and AUPR  
  - AUC and AUPR per edge type  

- **Visualization**
  - Plots of model performance  
  - Comparative analysis across models  

- **Optimization Results**
  - Hyperparameter tuning for model channels  
  - Selection of optimal configurations based on overall AUC  

- **Summary Files**
  - Aggregated results across all seeds  
  - Excel files with detailed performance breakdowns
